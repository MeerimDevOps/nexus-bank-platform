# Nexus Bank Platform

## Project Overview

Nexus Bank Platform is a secure and scalable AWS cloud infrastructure built with Terraform.

## Architecture

- AWS VPC
- Public and Private Subnets
- Internet Gateway
- NAT Gateway
- Application Load Balancer
- Amazon ECS on EC2
- Amazon RDS MySQL
- AWS Secrets Manager
- CloudWatch Monitoring
- CloudWatch Alarms
- ECS Auto Scaling

## Security

- Private application subnets
- Isolated database subnets
- Least privilege IAM
- Security Groups
- Secrets stored in AWS Secrets Manager

## Infrastructure as Code

All infrastructure is deployed using reusable Terraform modules.
